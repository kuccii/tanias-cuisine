# Tania's Company Profile Presentation — Design Spec

**Date:** 2026-07-12
**Status:** Approved
**Output:** 5-slide Canva presentation (dark theme, gold accents)

## Audience
All-purpose company profile — website visitors, walk-in clients, corporate partners, event planners.

## Core Takeaway
Tania's is Rwanda's premium catering and hospitality brand — quality-first, modern hospitality, best-in-class execution.

## Design System
- **Background:** Dark (near-black, matching taniascuisine.rw website)
- **Accent:** Gold/amber (`var(--primary)` equivalent) for headlines, key words, borders
- **Typography:** Display font for headlines (bold), clean sans-serif for body
- **Imagery:** Venue photos, food plates, catering setups — low opacity on background slides, full bleed where appropriate
- **Tone:** Confident, warm, premium — not salesy
