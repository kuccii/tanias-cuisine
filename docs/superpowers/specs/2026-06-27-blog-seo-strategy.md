  - Add "Review us on Google" QR card to every bill
  - Train waitstaff to ask happy tables
  - Follow up with catering clients 48h post-event
  - Never incentivize or fake reviews
- Fix NAP inconsistencies:
  - RDB listing shows 0788500635 / t.gakuba@gmail.com — must match website: +250 789 289 450 / hello@taniascuisine.rw
  - Audit ALL directories for NAP consistency

**Tier 2 — Supporting Signals:**
- Upload 3-5 fresh photos weekly (food, ambiance, events, staff)
- Post to GBP at least once a week (specials, events, new items)
- Populate Q&A section with 5-6 common questions

**Tier 3 — Blog's Role in Local Pack:**

## 6. Technical Architecture

### 6.1 Approach: Markdown-in-Repo

RECOMMENDED over headless CMS (over-engineered for AI-written content) or MDX (overkill for text+images).

**Dependencies:**
- `gray-matter` — parse YAML frontmatter
- `marked` — convert markdown to HTML

### 6.2 Directory Structure

```
src/content/blog/
├── corporate-catering-kigali.md
├── twatundi-beef-guide.md
├── gishushu-restaurant-guide.md
└── ...

src/routes/
├── blog/
│   ├── index.tsx          →  /blog  (listing, 9/page)
│   └── $slug.tsx          →  /blog/:slug  (single post)

src/lib/blog.ts             — markdown loader utility

public/blog/                — blog post images
```

### 6.3 Markdown Frontmatter Format

```yaml
---
title: "The Complete Guide to Corporate Catering in Kigali"
slug: corporate-catering-kigali
description: "Planning a corporate event in Kigali? From buffet logistics to per-person budgets, this guide covers everything you need to know about corporate catering."
date: 2026-06-27
author: "Tania's Cuisine & Lounge"
category: "events"
tags:
  - corporate catering
  - Kigali events
  - event planning
image: /blog/corporate-catering-kigali.jpg
readingTime: 6
published: true
---
```

### 6.4 URL Structure

Flat: `taniascuisine.rw/blog/corporate-catering-kigali`
NOT: `taniascuisine.rw/blog/2026/06/corporate-catering-kigali`

### 6.5 Blog Data Utility (`src/lib/blog.ts`)

Core loader using Vite's `import.meta.glob` (compile-time). Exports:
- `getAllPosts()` → sorted by date desc
- `getPostBySlug(slug)` → full post with HTML content
- `getPostsByCategory(cat)` → filtered for sidebar/related

All synchronous, zero runtime cost.

### 6.6 Routes

**Blog listing (`/blog`):**
- `head()`: title "Blog — Tania's Cuisine & Lounge | Kigali Dining Guides"
- Grid of post cards (9/page, paginated)
- Each card: image, title, date, category badge, excerpt
- Breadcrumbs: Home > Blog

**Blog post (`/blog/:slug`):**
- `head()`: dynamic meta from frontmatter
- Injects Article JSON-LD schema (see 6.8)
- Breadcrumbs: Home > Blog > Post Title
- Renders: featured image, title, metadata bar, HTML content, related posts, CTA
- 404 handling for unmatched slugs

### 6.7 Internal Linking

Per blog post:
- 1 link to homepage (branded anchor: "Tania's Cuisine & Lounge")
- 1-2 links to service pages (exact-match anchor text)
- 1 link to related blog post
- 1 link to contact/reservations page

DO NOT: use "click here", link every post to every page, or reuse same anchor text for different targets.

### 6.8 Article Schema Template

```json
<script type="application/ld+json">
{
  "@context": "https://schema.org",
  "@type": "Article",
  "headline": "{{POST_TITLE}}",
  "description": "{{META_DESCRIPTION}}",
  "image": "{{FEATURED_IMAGE_URL}}",
  "author": { "@type": "Person", "name": "Tania's Cuisine & Lounge" },
  "publisher": {
    "@type": "Organization",
    "name": "Tania's Cuisine & Lounge",
    "logo": { "@type": "ImageObject", "url": "https://taniascuisine.rw/assets/logo-CGNwafV6.png" }
  },
  "datePublished": "{{DATE}}",
  "dateModified": "{{DATE}}",
  "mainEntityOfPage": { "@type": "WebPage", "@id": "https://taniascuisine.rw/blog/{{SLUG}}" }
}
</script>
```

### 6.9 Content Length Guidelines

| Competition Level | Min Words | Recommended |
|---|---|---|
| Zero competition (unique dish names) | 800 | 1,000-1,500 |
| Low-Medium ("corporate catering Kigali") | 1,500 | 2,000-2,500 |
| Competitive ("best African restaurants Kigali") | 2,000 | 2,500-3,000+ |
| Link-bait / guide | 2,500 | 3,000-4,000 |

### 6.10 Image Optimization

- Format: WebP with JPEG fallback
- Featured images: < 200KB, inline: < 100KB
- Alt text: descriptive with keyword once (not stuffed)
- Filenames: `tanias-cuisine-corporate-catering-kigali.jpg` (hyphenated, descriptive)
- HTML: `loading="lazy"`, explicit `width`/`height`

### 6.11 Sitemap

Approach A (Recommended): Dynamic sitemap route at `src/routes/api/sitemap.xml.ts` that reads blog posts and generates complete XML including all blog URLs.

### 6.12 Performance

Automatic code splitting via Vite + TanStack Router (each route is a separate chunk). Blog routes do not bloat main bundle.

## 10. Risks

1. **NAP inconsistency across web** — RDB listing shows old phone/email. Fix immediately.
2. **Review stagnation** — Negative reviews without responses hurt. Respond professionally to all.
3. **Thin/self-promotional blog content** — Content must genuinely help the reader, not just sell.
4. **Ignoring GBP while focusing on blog** — Blog alone won't move Local Pack. Do both.
